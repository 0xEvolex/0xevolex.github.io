Introduction

The primary goal of this website is to provide a simple and easy-to-use interface for tracking events on the Sailor Online Persian Empire server (125 Cap). It tracks all events in real-time and translates them into your timezone. Additionally, it will provide you with sound alerts for upcoming events, even while the browser is minimized. So you'll never miss an event again!

 Allows to mute/unmute and adjust the volume of the sound.
Sets the time in which the event will be announced before it starts.
Allows you to select the voice for text-to-speech announcements. These are browser and/or system dependent.
Show/hide events from the table. Hidden events will not be announced anymore.
Allows you to select the language for the website. Still in development.
Table
  • Event Name: The name of the event.
  • Remaining Time: Time until the event starts or ends (if active). Orange indicates the registration period is running.
  • Register Method: The method to register for the event.
  • Next Event Date: The date and time of the next event.
  • Sound: The sound settings for this specific event.
